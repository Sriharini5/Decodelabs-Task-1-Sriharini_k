# DecodeLabs Task 1 - Data Cleaning & Preparation

## Project Overview
This project focuses on cleaning and preparing a raw dataset for analysis. The dataset was processed to improve data quality by handling missing values, removing duplicates, and standardizing data formats.

## Objectives
- Identify missing values
- Remove duplicate records
- Standardize data formats
- Validate data quality
- Prepare an analysis-ready dataset

## Tools Used
- Python
- Pandas
- Google Colab

## Files Included
- Raw Dataset
- Cleaned Dataset
- Data Cleaning Notebook
- Screenshots of Cleaning Process

## Data Cleaning Steps
1. Checked for missing values
2. Removed duplicate records
3. Standardized column formats
4. Validated data consistency
5. Saved the cleaned dataset

## Outcome
Successfully transformed raw data into a clean, reliable, and analysis-ready dataset suitable for future data analytics tasks.
